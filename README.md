# Django SaaS Demo

## Setup
```
make build
make migrate

-> Access link http://127.0.0.1:8000/
```

### start after each stop
```
make start
```

### migrate
```
make migrate
```

### down
```
make down
```

### bash
```
make bash
```

### run server
```
Access link http://127.0.0.1:8000/
```

### Create SECRET_KEY
```
make secret-key
```
